Python

Python is an interpreted, high-level, general-purpose programming language.

It supports multiple programming paradigms.

  • Created by Guido van Rossum and first released in 1991.
  • Emphasizes code readability with its notable use of significant whitespace.
  • Supports multiple programming paradigms, including procedural, object-oriented, and functional programming.
print("Python is versatile!")
Hello world

One of the simplest programs in Python is the "Hello, World!" program.

This program is often used as a first step in learning a new programming language.

print("Hello, World!")

This program outputs the text "Hello, World!" to the console.

Variables

Variables in Python are used to store information that can be referenced and manipulated in a program.

They are fundamental to programming and allow for dynamic data handling.

  • Python variables do not need explicit declaration to reserve memory space.
  • The declaration happens automatically when you assign a value to a variable.
  • Example: my_variable = 10
my_variable = 10
